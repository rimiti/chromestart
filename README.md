[![License][license-badge]][license]

## Description

This package aims to hightly improve the way you start your chrome browser. It allows you to start your browser with a specific profile, a specific url, and a specific window size.

## Install

```
yarn add @rimiti/chromestart
```

## Scripts

Run using yarn run `<script>` command.

    clean       - Remove temporarily folders.
    build       - Compile source files.

    lint        - Lint source files.

    test        - Runs all tests with coverage.
